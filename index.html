What is ViennaDotNet?

Minecraft Earth shut down in 2021 — but we brought it back. ViennaDotNet is an unofficial port of Project Vienna to .NET, adding new features like Shop, Map rendering, and an Admin panel.

How to Start Playing?

1

Set Up the Server

Clone the ViennaDotNet repository, install .NET 10.0 SDK and Java 17, then run the publish and launcher scripts.

2

Configure & Start Server

Open localhost:5000, create an account, set your PC's IP address in Server Options, and click Start.

3

Patch the Client

Use MCE Patcher or Project Earth Patcher to patch your Minecraft Earth APK with your server IP.

4

Launch the Game!

Install the patched APK on your device and start exploring Minecraft Earth again!

FAQ

What are the requirements?
You need a PC running Windows or Linux, .NET 10.0 SDK, and Java 17 (JRE or JDK). macOS is untested.
What is ViennaDotNet?
ViennaDotNet is an unofficial port of Project Vienna to .NET. It adds new features like Shop, Map rendering, and an Admin panel. More info at github.com/Earth-Restored/ViennaDotNet.
App says "Cannot connect to the network"?
Make sure the server is running, check your PC's IP address is correct, verify your firewall isn't blocking it, and ensure your phone and PC are on the same network.
Where to download the APK?
You need a legal copy of Minecraft Earth. We do not provide pre-patched APKs - use the MCE Patcher or Project Earth Patcher to patch your own APK.
